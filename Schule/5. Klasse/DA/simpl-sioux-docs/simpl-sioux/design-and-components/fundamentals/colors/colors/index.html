    Colors

    General

    The SiOUX design system provides a light and a dark theme. Each of the themes consists of the following color categories:

    • Primary color palette
    • Accent color palette
    • Functional colors
    • Supporting colors

    Primary Color Palette

    The primary colors are used throughout the complete app.

    Accent Color Palette

    The accent color is used rarely, especially to give attention to specific user interface elements.

    Functional Colors

    The functional colors are used to present status information. They are identical for both themes except the functional gray. You can retrieve the functional colors by means of the following function:

    get-functional-colors($theme)

    The following functional colors are defined:

    Dark Theme vs. Light Theme

    The choice of the theme can be influenced by many points such as the usability, the business goals, the current design trends, the target audience and more. If the target audience are mainly young people, a dark user interface seems to be more attractive, while middle-aged and older people prefer light color themes.

    From usability point of view the decision should not be due to pure aesthetic reasons. Instead the decision should depend on the customer and his working conditions. The following aspects can support you in your decision:

    Readability and Amount of Usage

    In general, black text on white background which is used in SiOUX Light Theme is the most readable. Nevertheless, it can fatigue the eyes if an application is used for a long time. For this reason, interfaces which are used intensively and provide a lot of text should use a light text on a dark background which is the case for the SiOUX Dark Theme. This is almost as readable, but less tiring.

    User Environment and Time of Use

    Regarding this criteria it is relevant in which room the application is used and during which time of the day. Dark interfaces are ideal during night time or in the evening, while light interfaces are more suitable during daytime. Applications which are used throughout the entire day should provide switchable themes where the customer can decide which theme to use.

    Use When SiOUX Light Theme

    1) The application is mainly used during day time in a bright environment
    2) The application is used for a short time and not extensively
    3) A lot of text is presented which requires a good readability
    4) The target audience are mainly middle-aged or older people

    Use When SiOUX Dark Theme

    1) The application is mainly used in the evening or in a dark environment
    2) The application is used extensively
    3) The target audience are young people

    Base Color

    As the elevations in the dark theme are shades of gray the dark theme consists additionally to the other four color categories of a base color. The base color of the SiOUX dark theme is #11151E. This is a dark gray which has been defined on base of the by Google Material Design recommended surface color #121212 and an 8% overlay witd its 700 tone of the accent color. Google Material Design suggests to use the accent color instead. Just as our accent color is quite light we used the 700 tone value to receive a darker base color.
