    Switch

    Design

    Switches allow users to switch an option between "on" or "off" (or analog opposite values).

           

           

    Usage

    Switches are used for single options, which can be defined as "on" or "off". Especially useful in option sheets, where settings are adjusted.

    Use When

    • you want to toggle a single item on or off, especially on mobile and tablet
    • something is immediately to be activated or deactivated (i.e. switching loudspeaker on or off)
    • each item in a set can be independently controlled

    Use Alternatively

    • radio buttons or an exposed dropdown menu, if the options available are mutually exclusive
    • a exposed dropdown menu, if users need to select one item from a long list (> 7 options) or a list with varying items
    • checkboxes, if the options available are not mutually exclusive.

    Anatomy

    • A switch is displayed with a track and a knob moving on this track between left and right end
    • For label definition see Wording.

    Placement

    • Switches are listed vertically as a group, vertically well aligned
    • A label is shown on the left hand side
    • The labels are left aligned.

    Behavior

    • Users tap it to toggle to an opposite value.
    • When a user toggles a switch, its corresponding action takes effect immediately
    • If a switch cannot be turned on, the switch will automatically turn back off.
    • Switches can either be:
      • On: the associated value or property is immediately activated, the knob is on the right end position on the track, the track and the knob highlighted (accent color)
      • Off: the associated value or property is deactivated, the knob is on the left end position on the track, the track and the knob neutrally colored
    • If it takes longer until the final state is confirmed, a progress indicator is displayed on the knob during the process
    • Wherever possible in the use context, define the switch(es) states by default.
    • Switches can have the states:
      • enabled
      • hover
      • focused
      • pressed.
