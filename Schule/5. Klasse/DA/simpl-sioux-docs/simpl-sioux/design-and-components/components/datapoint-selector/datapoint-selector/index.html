    API reference for Sioux Datapoint Selector

    import {SiouxDatapointSelectorModule} from '@simpl/sioux-ng';

    Directives

    Sioux Datapoint Selector

    Selector: sioux-datapoint-selector Exported as: SiouxDatapointSelectorComponent

    Name Description
    @Input() identifier: string Contains the name (identifier), which is stored in local storage.
    @Input() data : DataPoint[] Contains the data to be loaded.
    @Input() loading= false Denotes whether the content is loading or not. By default, it is 'false'.
    @Input() disabled= false Disables the click property on the datapoint selector. By default, it is 'false'.
    @Input() nodatapoints: string Contains the string which will get populated when the table dont have any data.
    @Input() readMode: StorageMode
    @Input() writeMode: StorageMode
    tableFilters: any The filtered set of data that has been matched by the filter string, or all the data if there is no filter.
    filter: string Filter term that should be used to filter out objects from the data array
    canFilter: boolean return true when the length of the datapoint is greater than 0.
    tableHeader: string Text label that should be used for the column header
    sorting: Sort Contains two properties "active", which denotes the type of method and "direction" which takes the value 'desc' or 'asc'. By default, active: 'data_point_text', direction: 'asc'.
